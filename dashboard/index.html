How to use
1. In the Wix Editor, click Add Elements → Apps → Repeater Likes.
2. Drag the Like Button widget inside any Repeater item.
3. Publish your site — visitors can now like any item. ♥

Like data is stored directly in your site's CMS collections (ItemLikes and LikeCounts) — visible in your Content Manager.
